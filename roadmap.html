🚀 Roadmap

ENSv2 Marketplace™ is building the curated ENS infrastructure namespace registry — a purpose-built, single-operator portfolio of 2,500+ protocol-grade names organized for enterprise subdomain leasing when ENSv2 launches on Ethereum L1.

This is a curated registry, not an open marketplace. Every name is hand-selected, scored, categorized, and prepared for lease. That quality control is the product.

✅ Complete

Phase 1 — Foundation (2025)

  • 2,500+ namespace portfolio built — Protocol-grade ENS names across 30 infrastructure verticals, acquired at $5/name with strict selection criteria
  • 30-category taxonomy developed — 260+ subcategories with a 1–15 scoring algorithm covering tier labels and lease rate pricing
  • Hardware wallet security — All names secured on Ledger Nano with verified ownership. Recovered from an early malware incident; security practices hardened
  • Brand & trademark established — ENSv2 Marketplace™ registered with the USPTO. ensv2marketplace.com live across 12 pages
  • Python tooling built — Custom scorer, category matcher, description generator, and analytics workbook covering the full portfolio
  • ENSv2 architecture research — Confirmed February 2026: ENSv2 stays on Ethereum L1 with a new roles system replacing the old fuses model
⚡ Active Now — 2026

Phase 2 — On-Chain Registry Completion

The current focus is setting all 6 on-chain ENS text records for every name in the portfolio — completing the registry infrastructure that makes the namespace catalog machine-readable and lease-ready.

  • On-chain text records — Setting 6 records per name across all 2,500+ names via Ledger. ~50–70 names per day. ETA late April 2026
  • Infrastructure map launched — New visual stack map showing all 30 verticals organized by protocol layer, with namespace authority section highlighting core primitives
  • Ongoing S-tier acquisition — Continuing to identify and register category-defining primitive names as they surface
  • Subdomain leasing available now — Manual direct leasing arrangements for enterprises who need custom subdomains before ENSv2 automated infrastructure launches
📍 Next — Late April / May 2026

Phase 3 — Registry Architecture Upgrade

Once all on-chain records are set, two internal architecture projects will modernize the platform:

  • Terminology refresh — Site-wide language update across all 12 pages: categories become verticals, subcategories become segments. Positions the registry correctly for a developer and enterprise audience
  • Schema registry redesign — Replace the current organic code system with a hierarchical dot-notation schema that mirrors ENSv2 subdomain paths (e.g. INFRA.BLOCKCHAIN.L2.BASED). Makes the catalog machine-readable and directly maps to lease structures
  • Enterprise outreach begins — Direct outreach to Web3 protocols, DAOs, and enterprise teams who are candidates for subdomain lease agreements
🔮 ENSv2 Launch — Mid 2026

Phase 4 — Automated Subdomain Leasing

When ENSv2 launches on Ethereum L1, the registry transitions to fully automated lease infrastructure:

  • ENSv2 migration — Migrate all names to ENSv2 hierarchical registries on Ethereum L1, leveraging the new roles system for trustless subdomain management
  • Automated leasing contracts — Smart contract-based subdomain leasing with stablecoin payments accepted from any EVM chain. No bridging required
  • Self-service leasing portal — Enterprises browse the registry, select a namespace, and deploy a subdomain lease directly on-chain with verifiable guarantees
  • First lease agreements closed — Convert outreach pipeline from Phase 3 into signed on-chain lease arrangements
🔮 Scale — 2027

Phase 5 — Enterprise Scale & API Integration

  • API access — Enable protocols and platforms to query the registry and integrate subdomain leasing directly into their own products
  • Bulk enterprise packages — Corporate lease arrangements covering multiple subdomains across related verticals with volume pricing
  • Cross-industry expansion — Target non-crypto enterprise adoption: healthcare, finance, logistics, legal — industries with identity and credentialing infrastructure needs
  • 20-year renewal guarantees — Formalize long-term lease agreements backed by the guaranteed renewal commitment already offered to lessees

Operating Principles

  • Curated, not open — Every name in the registry is hand-selected. Quality control is the product. No third-party listings
  • Infrastructure first — Names that solve real protocol, identity, and enterprise problems — not speculation or lifestyle brands
  • Lease, don't sell — Recurring revenue model. Subdomains leased at rates tied to tier scoring ($750–$10,000/yr). Parent names are never sold
  • Single operator, maximum security — All names held on Ledger hardware wallet by one verified operator. No shared custody, no delegation risk
  • Built for ENSv2 — The entire portfolio was architected around the hierarchical subdomain system ENSv2 is delivering on Ethereum L1
